# 🌾 AgroLink – AI-Powered Agriculture Support Platform

AgroLink is a full-stack web application designed to support farmers by providing AI-powered crop recommendations, weather information, and a digital marketplace for agricultural products. The platform aims to connect farmers with buyers while improving decision-making through modern technology.

> **Note:** The current live demo showcases the frontend interface. Backend services were developed separately and are not deployed in the current demo.

## 💻 Installation

### Clone the Repository

```bash

```

### Frontend

```bash
cd client
npm install
npm start
```

### Backend

```bash
cd server
npm install
npm start
```
